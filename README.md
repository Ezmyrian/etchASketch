# etchASketch
5/13/24

In this project I will be creating a browser based program akin to etch-a-sketch.
I will be using javascript to create the divs and flexbox to make them appear as a grid.  Once the divs are created a hover effect will change the color of the div as the mouse passes over them.  A button will present the user with a popup asking the user for the number of squares per side for a new grid.  The new grid will remove the old grid and take up the same amount of total space.  

As a bonus, I will code the color used for each square to be a random RGB value with each interaction.  I will then code in a progressive darkening effect where each interaction darkens the square by 10%.  After a square has been interacted with 10 times it will be black.
